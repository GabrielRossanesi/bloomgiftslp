# Bloom Gifts Landing Page + Catálogo - Proposta Comercial

Proposta comercial interativa em Next.js para apresentar a nova landing page da Bloom Gifts com catálogo de produtos, seleção de quantidade e direcionamento personalizado para WhatsApp.

## Stack

- Next.js App Router
- React + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Comandos

```bash
npm.cmd install
npm.cmd run dev
npm.cmd run build
```

## Estrutura

- `src/app`: layout, estilos globais e rota principal.
- `src/components`: componentes da proposta.
- `src/lib`: dados estruturados do conteúdo comercial.
